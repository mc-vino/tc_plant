# tc_plant

Проект сайта.

## Структура веток

- **`main`** — стабильная, деплоящаяся ветка. Прямые коммиты не приветствуются.
- **Фича-ветки** (`feature/...`, `fix/...`) — ответвляются от `main`, вливаются обратно через pull request.

## Начало работы

```bash
# установка зависимостей (после добавления package.json)
# npm install

# запуск в режиме разработки
# npm run dev
```

## Инструменты

- **Context7 MCP** — сервер для подтягивания актуальной документации библиотек.
  Настроен в `.mcp.json`; при первом открытии проекта в Claude Code потребуется
  подтвердить доверие к серверу.
